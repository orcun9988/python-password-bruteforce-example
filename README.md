# Python Password Brute-Force Example

> ⚠️ **Educational only.** This repository is archived and exists only as a reference for security awareness and CS coursework. Do not use against any account or service you do not own or have explicit written permission to test.

Simple Python example that demonstrates the general structure of a password-guessing loop against a sample wordlist. The script does **not** make real network requests; the login function is a placeholder.

## Usage

```bash
python example.py <username> <wordlist.txt>
```
